# Apple Container Skill

This repository contains the `apple-container-skill` Agent Skill. This skill allows agents to interact with the Apple Container CLI to manage containers, images, volumes, networks, and system services on macOS.

It complies with the [Agent Skills specification](https://agentskills.io/specification).

## Usage

To use this skill, import the `SKILL.md` file into your agent's configuration.

## Features

- **System Management**: Start/stop container services, manage DNS, and system properties.
- **Container Lifecycle**: Run, create, start, stop, kill, and delete containers.
- **Image Management**: Build, pull, push, list, and delete images.
- **Volume & Network Management**: Create and manage volumes and networks.
